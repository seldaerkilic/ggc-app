Pick Your Game!

This website is about an imaginary board game tournement which is organized since 2019. Being an imaginary event site, the page is full of fictional references from Star Wars to Doctor Who. Still, the players section has some external links about the characters.

The whole purpose of this project is to implement basic HTML, CSS, and some JavaScript concepts to simulate a multi-page website. The MVP of the project includes a header, banner image, a table with minimum 5 rows, and a footer.
